# Voyages

A modern travel agency website built with Next.js, TypeScript, and Tailwind CSS.

## Project Structure

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS, shadcn/ui (for UI components like the navbar)
- **Animation**: Motion (formerly Framer Motion)

## Features

- Responsive navigation with dropdown menus
- Clean and light theme
- Complete site structure for a travel agency (Tours, Stays, Cruises, etc.)
- Home page with hero section
- Footer with legal links and copyright
